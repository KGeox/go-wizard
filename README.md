## Commands
#### 2D:
- a / left_key = move left
- d/ right_key = move right
- space = jump
- Tab = Interact with environment

#### 3D:
- w = forward
- a = left
- s = backward
- d = right
- Tab = interact
- i = attack
- space = jump
